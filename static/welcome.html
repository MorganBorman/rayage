Welcome to Rayage.

To create a new project from scratch or load a provided template, click on 'Project', then 'New Project'.

To open an existing project, click on 'Project', then 'Open Project'.

Options to delete the project, close the project, add a file, save the file, or delete the file are also under the 'Project' dropdown.

The 'Edit' dropdown includes options to undo, redo, copy, paste, cut, and select all.

The 'Build' button builds your project and the 'Run' button runs it. You must build your project before you can run it.

The 'Logout' button will log you out of Rayage.
